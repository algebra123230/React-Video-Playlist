## ✨ Features

- Save video playlist (localstorage).

- Dark / Light mode.

- Easy to manage video playlist.

- Autoplay / loop videos in the playlist.

# ⚙️ Usage

1. Clone the Repo
2. Run `npm install`
3. Run `npm start`
